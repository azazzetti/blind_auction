# blind_auction
A simple command-line silent auction program where users submit bids, and the highest bidder is determined and announced.

This project is a command-line based silent auction program where multiple users can place bids. Each user provides their name and bid amount, and after all bids are submitted, the program determines the highest bidder and announces the winner.

Features:
Users can continuously submit bids until no more participants remain.
Automatically calculates and displays the highest bid and the winner.
Clears the screen after each bid to maintain confidentiality.
Simple and easy-to-use bidding interface.
